# Privacy Hosting Guide (GitHub Pages)

1. Push this repository to GitHub.
2. Open repository settings: `Settings -> Pages`.
3. In `Build and deployment`, select:
   - `Source`: Deploy from a branch
   - `Branch`: `master`
   - `Folder`: `/docs`
4. Save and wait until Pages is published.
5. Verify URLs:
   - `https://pavloiv.github.io/naked_truth/privacy_index.html`
   - `https://pavloiv.github.io/naked_truth/support.html`

These two links are used in App Store Connect fields:
- Privacy Policy URL
- Support URL
